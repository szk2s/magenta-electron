# MAGENTA-ELECTRON

**NOTE: You can run this demonstration only when you are online.**

## Getting started
### Create virtual environment

```
python3 -m venv venv
```
### Activate venv
```
source venv/bin/activate
```
### Install python packages
```
pip install -r requirements.txt
```

### Install node modules
```
yarn
```

### Run demo
```
yarn start
```
